Hello Manjaro!

最后更新于：4 天前

​ 即使是 MacType ，也只能优化 Windows 中一部分的字体渲染。部分软件（如 AIMP 和使用 DirectWrite 的 Chrome ）的显示依旧很糟糕。同时我也饱受 Windows 奇怪的逻辑（设置和控制面板里乱七八糟），于是打算转向 Linux 。

​ 设备：联想小新-14 2019（AMD平台：API版）

处理器: AMD 3500U
显卡: Vega 8
内存: 8GB DDR4 2400MHz
显示器: 1920x1080
网卡：Realtek 8821CE

​ 本来希望安装 Ubuntu 20.04 ，制作 LiveCD 后进入，不能免驱使用网卡，且 Ubuntu 基于 Debian , Debian 据说硬件兼容性较差，遂放弃了 Debian 系（后来发现可以使用 deepin ）。Red Hat 系还是适合服务器，桌面上放弃。于是就剩下了 Arch 系。 Archlinux 配置过于繁琐，放弃。而 Manjaro 在 LiveCD 中居然能免驱 rtl8821ce ，就只能选择它了。

​ 本来安装的是 Manjaro Gnome （Gnome 的暗色实在是太好看了），可惜 Gnome 在我这里出现了各种各样的问题，包括但不限于终端打不开、 Firefox 启动需要非常非常非常长时间等问题（大概是因为我太菜了）。不得已使用了 KDE 。至于如何安装就不再赘述。

​ 简单配置一下之后， KDE 也并没有我想象的那么糟糕。有些快捷键和 Windows 一样，适应比较快。默认的字体某些字形会很奇怪，但渲染比 Windows 10 好了不知道多少。字体设置里把所有字的字重都加重 1pt 之后很合我胃口。 pacman 和 AUR 用起来也是非常舒服的。

先写到这里，想起来再写。
